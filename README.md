zsh-antigen
===========

Antigen-based zsh configuration. Clone this into ~/.zsh-antigen, and then symlink the zshrc file to ~/.zshrc:

```
git clone https://github.com/mgdm/zsh-antigen.git ~/.zsh-antigen
cd
ln -s .zsh-antigen/zshrc .zshrc
```
